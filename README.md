# SSE-Home-Service-App
Secure Software Engineering App

This is a home services App where users can book for house related services. 


## Run this project

### Using Docker.

- Install Docker 
- run docker
- Run this command 
        ```docker-compose up```

- Go to localhost:8000 to view the project

### Requirements

- Mysql >=8.0
- Python >= 3.11
